Логотип

В основе метафоры логотипа использован "слэш" - один из базовых элементов интернет-адреса. Он работает как символ вложенности, обозначая стратегию.

Логотип создан на основе главного логотипа компании.

Сетка

Каждый макет разработан на основе сетки. Вы можете скачать основные сетки для нескольких популярных макетов, чтобы разработка проходила легче.

Охранная зона

Логотипу необходим "воздух". Для разных ситуаций используется разный тип логотипа. Не стоит пренебрегать пространством вокруг логотипа.
